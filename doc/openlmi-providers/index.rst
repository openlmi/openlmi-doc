OpenLMI providers
=================

This document includes detailed description about CIM providers distributed
in openlmi-providers package, see below for full list.

See our `overview <http://www.openlmi.org/node/1785>`_ for details what is
a provider and how the whole CIM+WBEM infrastructure is supposed to work.

Other OpenLMI providers, not covered in this documentation, are
:ref:`networking <openlmi-networking-provider>` and
:ref:`storage <openlmi-storage-provider>`.
